📄 כלי הגהת דוחות כספיים

סקיל של Claude לבדיקת תקינות דוח כספי לפני שליחתו ללקוח: כתיב, דקדוק, טרמינולוגיה, מבנה, גילויים נדרשים ואימות מספרי מלא. הפלט הוא קובץ Excel מסודר עם שלושה גיליונות בעברית.

🤖 זהו סקיל של Claude, לא כלי דפדפן. מורידים את החבילה ומטעינים אותה ל-Claude (Claude Code או Claude Desktop). Claude קורא את עמודי הדוח ומפיק את דוח ההגהה.

הורדה

הורדת החבילה (ZIP)

החבילה כוללת את ההנחיות, הסקריפטים והתיעוד. גנרית לחלוטין, ללא נתוני לקוח.

⚠ נדרשים: Claude (Claude Code או Claude Desktop) עם יכולת ראייה, ו-Python 3 מותקן. פעם אחת יש להריץ: pip install pymupdf openpyxl

למה הכלי מיועד

הכלי עובר על הדוח הכספי עמוד אחר עמוד ומאתר שגיאות טרם השליחה ללקוח. הוא קורא את העמודים ויזואלית (כתמונות), משום שחילוץ טקסט מ-PDF עברי אינו אמין: אותיות נשמטות משכבת הטקסט (במיוחד האות נ') ואותיות דומות מתבלבלות. לצד בדיקת הניסוח, הכלי מאמת באופן עצמאי כל סכום וסיכום ביניים בדוח.

כתיב ודקדוקטרמינולוגיה חשבונאיתמבנה וסדר ביאורים גילויים נדרשיםאימות מספריעסק חי

הפלט: הגהה_<לקוח>_<שנה>.xlsx עם גיליונות סיכום, פירוט ממצאים ואימות נתונים מספריים. זו טיוטה לבדיקת אדם לפני שליחה ללקוח.

איך משתמשים

  1. Claude Code: מחלצים את התיקייה ומעתיקים אותה אל ~/.claude/skills/ (ב-Windows: C:\Users\<שם>\.claude\skills\).
  2. Claude Desktop: מעלים את קובץ ה-ZIP תחת Settings > Capabilities / Skills.
  3. פותחים שיחה בתיקייה שבה נמצא ה-PDF של הדוח, ומבקשים למשל: תעבור הגהה על הדוח.
  4. Claude מרנדר את העמודים, קורא אותם, מאמת את המספרים, ומפיק את קובץ ה-Excel לצד הדוח.
⚠ הליבה החכמה, קריאת העמודים, מתבצעת על ידי Claude. הרצה מקומית של הסקריפטים לבדה מבצעת רק את החלקים המכניים (רינדור ויצירת ה-Excel), לא את הקריאה והשיפוט.

דרישות והתקנה

Claude עם יכולת ראייה (Claude Code או Claude Desktop) ו-Python 3 מותקן. אם אין Python על המחשב, מתקינים אותו מהמקור הרשמי:

הורדת Python (python.org, המקור הרשמי)

בהתקנה יש לסמן Add python.exe to PATH. לאחר מכן מריצים פעם אחת בשורת הפקודה:

pip install pymupdf openpyxl

הוראות מפורטות בתוך החבילה: INSTALL.md ו-README.md
