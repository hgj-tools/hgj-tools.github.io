# כלים שימושיים - HGJ

אוסף כלי עזר חינמיים למשרדי רואי חשבון, מבית הוגן, גינזבורג, יודלביץ ושות', רואי חשבון.

**האתר:** https://hgj-tools.github.io/

## הכלים

| כלי | קטגוריה | סוג |
|---|---|---|
| [כלי סיור - הגנת הפרטיות](https://hgj-tools.github.io/privacy-survey/) | אבטחת מידע והגנת הפרטיות | דפדפן (PWA) |
| [כלי השחרת נתונים](https://hgj-tools.github.io/data-masking/) | אבטחת מידע והגנת הפרטיות | הורדה (Windows + Python) |
| [הכתבה קולית בעברית](https://hgj-tools.github.io/dictation/) | כלים כלליים | דפדפן (Chrome / Edge) |

## עקרונות

- **עיבוד מקומי:** כלים שמעבדים קבצים או נתונים עושים זאת בדפדפן או במחשב של המשתמש בלבד. שום נתון לא נשלח לשרת, אלא אם צוין אחרת במפורש בעמוד הכלי.
- **ללא מעקב:** אין באתר עוגיות, אנליטיקס או טעינת משאבים מצד שלישי.
- **אין להזין לכלים פרטי לקוחות או מידע רגיש.**

פרטים מלאים: [הצהרת נגישות, פרטיות ותנאי שימוש](https://hgj-tools.github.io/policy/).
