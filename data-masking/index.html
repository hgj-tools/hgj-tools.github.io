🛡 כלי השחרת נתונים

כלי מקומי (אופליין) להשחרת קובצי Excel בלבד לפני שיתופם לניתוח, עם שחזור מלא של המזהים בסוף התהליך כדי לזהות למי שייך כל ממצא.

📊 Excel בלבד (.xlsx / .xls). הכלי משחיר עמודות בטבלה. קובצי Word ו-PDF אינם נתמכים.

הורדה

הורדת הכלי (ZIP)

חילוץ התיקייה למחשב, ולחיצה כפולה על הפעלה.bat. הכלי רץ מקומית בלבד, שום נתון לא נשלח לרשת.

⚠ נדרש Python 3 מותקן על המחשב. אם הוא חסר, הכלי יפתח את עמוד ההורדה של Python. בהתקנה יש לסמן Add python.exe to PATH, ואז להריץ פעם אחת: pip install pandas openpyxl xlrd cryptography

למה הכלי מיועד

הכלי מחליף מזהים אישיים (שם, ת"ז, חשבון, טלפון, כתובת, מעסיק ועוד) באסימונים עקביים והפיכים: אותו ערך מקורי מקבל תמיד את אותו אסימון בכל הקבצים, כך שההצלבות בין טבלאות נשמרות. את הקובץ המושחר אפשר לשתף בבטחה (למשל להעלות לניתוח), ובסוף התהליך משחזרים את המזהים האמיתיים מקומית בעזרת הסיסמה.

שם NAME-ת"ז ID-חשבון ACC- בנק BNK-טלפון PHN-כתובת ADR- מעסיק EMP-עמודה חופשית FLD-

קובץ Excel בלבד (.xlsx / .xls). קובצי Word ו-PDF אינם נתמכים.

איך משתמשים

  1. מחלצים את התיקייה ומריצים הפעלה.bat (נפתח מסך בית).
  2. השחרה: שמים את הקבצים המקוריים בתיקייה 1, בוחרים אילו עמודות להשחיר (שם/ת"ז נעולים תמיד), ומזינים סיסמה. הפלט המושחר נוצר בתיקייה 2.
  3. שחזור: שמים את קובץ הממצאים בתיקייה 3, מזינים את אותה סיסמה, והקובץ עם המזהים האמיתיים נוצר בתיקייה 4.
⚠ בהרצה הראשונה בוחרים סיסמה. יש לזכור אותה, בלעדיה לא ניתן לשחזר את השמות והת"ז בסוף התהליך.

דרישות והתקנה

Windows עם Python 3 מותקן. אם עדיין אין Python על המחשב, מתקינים אותו מהמקור הרשמי:

הורדת Python (python.org, המקור הרשמי)

בהתקנה יש לסמן Add python.exe to PATH. לאחר מכן, אם חסרות חבילות, מריצים פעם אחת בשורת הפקודה:

pip install pandas openpyxl xlrd cryptography

הוראות מפורטות בתוך החבילה: _קבצי מערכת (לא לגעת)/הוראות-שימוש.html

🔒 אבטחה: טבלת המיפוי נשמרת מוצפנת (Fernet, AES-128 + HMAC, מפתח נגזר מהסיסמה ב-PBKDF2 עם 390,000 איטרציות). הקבצים המושחרים עצמם אינם מכילים מידע אישי.
